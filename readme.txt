The TrackToAct Sensor 

== Description ==

TrackToAct VisiLeads is a lead generating plugin that lets you turn website visitors into contacts. This plugin adds TrackToAct VisiLeads smart sensor code on your pages. You simply enter tracking code and VisiLeads does the rest including activating the Autosuggest feature to increase conversions on existing web forms. See http://tracktoact.com/ for more info.

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress
